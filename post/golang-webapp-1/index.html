Complete Golang Web app: Introduction



Sat Feb 14, 2015
golang web development

So, I joined the line of backend developers who have tried out golang. To learn web development in golang, I set out to develop kaku.com.ng in golang. Along the way, I learnt: * To implement facebook authentication in golang * To do file storage on AWS from golang * To host a golang web app on heroku * To implement an autocompletion backend that returns JSON * To implement random naming of files to be uploaded * To use mongo db via the mgo * To implement pagination with a mongo db backend * To manage sessions in golang

Over the course of this series, I’d be sharing how i achieved all these. If you feel the methods I used could be improved on, please feel free to share your solutions in the comments.

Before I start, I’d give a background on the web app; kaku.com is a place and events discovery service. Any user can sign in using their facebook accounts and add a place. Other users can search for places using three search boxed. Users view these places and post reviews.

On the backend, I made use of no web framework; No martini, No gin, No Revel. I didnt even use any routing library. Turns out that net/http is more than capable, with a little extra effort. Coming from a python django background, it felt like relearning how the web works; Requests and Responses.

For Golang, I’ve ended up with a mindset that most external dependencies add thousands of lines to your codebase, and that is equivalent to extra effort maintaining and keeping track of, for features that you could have easily achieved with a little effort and a few extra lines of code. For example, routing; A little bit of string manipulation and regular expressions could save you huge dependencies.

My entire codebase is available at github/tonyalaribe/kaku-go for the golang backend and templates, but if you like how the frontend looks and want to see what’s happening behind the scenes, the HTML, SASS and Javascript files are available at github.com/tonyalaribe/kaku. Running bower install and npm install should install all the gulp files, converting it into the complete dev environment.

Feel free to clone the repositories and hack till your hearts content.
